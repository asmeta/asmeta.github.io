Asmeta is used in our language engineering model-driven process (Lemp)

Papers

The proposal of using Asmeta (together with several techniques) for language semantics can be found in:

A. Gargantini, E. Riccobene, and P. Scandurra. A semantic framework for metamodel-based languages. Journal of Automated Software Engineering (2009)[link]

Lemp applied to Finite State Machines

We have defined a process for defining the semantics of modelling languages. We have applied this process to the Finite State Machines and here you can find some artifacts:

  • the FSM metamodel
  • the EMFText cs grammar file
  • the Xtext grammar file
  • an example of FSM, evenFsm
  • the ATL iota transformation from FSM to ASM
  • the resulting ASM for the even example
  • an example of scenario for semantics validation

The svn repository contains many other files (xmi, EMF java api and so on).
