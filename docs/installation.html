Download and Installation

Current version: 0.9.2 (19 June 2007) - post ASM 2007

We have initially released our binary files on our web site. We plan to use the release system of SourceForge only later this year.

Command Line version

Prerequisites

Java: Asmeta tools need Java >= 1.5.0 to run.

Download

Download the parser: AsmetaLc.jar

Download the interpreter: AsmetaS.jar

How to run (see userdoc/tools.html for instructions)

To run the parser - in brief

java -jar AsmetaLc.jar <filename>
To run the ASM simulator - in brief 
java -jar AsmetaS.jar <filename>

Eclipse interface ASMEE

A graphical front-end called ASMEE (ASM Eclipse Environment) which allows to edit and manipulate AsmM models. This environment is implemented as an Eclipse plug-in.

Prerequisites

You need the last version of eclipse.

Option 1. Update Site

The update site for Asmee is http://svn.code.sf.net/p/asmeta/code/code/stable/asmee_update/. Perform the following steps to install Asmee from the update site:

1. From your Eclipse menu select Help → Software Updates → Find and Install...

2. Check Search for new features and press Next.

3. Press New Remote Site... and enter http://svn.code.sf.net/p/asmeta/code/code/stable/asmee_update/ for the update site URL:

4. Press Finish.

5. Follow the steps in the installation wizard.

Option 2. Manual Installation

Download asmee_site.zip which contains a zipped version of the asmee update site. Unzip it and copy it in a temporary directory (for example c:\temp\asmee). Then update eclipse like in Option 1, except that in step 3 choose new Local Site and select the directory (c:\temp\asmee) where you have unzipped the asmee_site.
